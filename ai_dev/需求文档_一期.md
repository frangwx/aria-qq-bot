# 绝区零 QQ 机器人需求文档（一期）

## 一、项目概述

### 1.1 项目背景
本项目旨在开发一款面向《绝区零》游戏玩家的 QQ 机器人，提供游戏资讯推送、智能对话、游戏信息查询等功能，提升玩家的游戏体验。

### 1.2 项目目标
- 构建一个稳定、可扩展的 QQ 机器人服务
- 提供绝区零游戏相关的资讯推送与查询功能
- 接入大模型实现智能对话，扮演游戏角色"爱芮"
- 支持多用户会话管理与长期记忆

### 1.3 参考项目
- **Miao-Yunzai**: 喵版 Yunzai，源自 Le-niao/Yunzai-Bot V3，提供机器人基础架构参考
- **ZZZ-Plugin**: 绝区零游戏信息获取插件，提供游戏数据接口参考

### 2.2 技术选型

| 模块 | 技术方案 | 说明 |
|------|----------|------|
| 机器人框架 | NoneBot2 | Python 异步框架，支持 OneBot 协议 |
| QQ 接入方案 | NapCat (OneBot v11) | 基于 QQ NT 客户端，更稳定 |
| 大模型接入 | OpenAI 兼容协议 | 支持 OpenAI、DeepSeek、Qwen 等 |
| 向量数据库 | ChromaDB | 用于对话记忆的向量化存储 |
| 关系数据库 | SQLite | 用户数据、配置存储 |

## 三、功能需求

### 3.1 设置相关

#### QQ 账户接入
- 支持 NapCat (OneBot v11) 协议接入
- 支持 WebSocket 连接
- 支持群聊/私聊消息收发

#### 主人 QQ 设置
- 通过配置文件设置主人账号
- 通过命令设置主人账号
- 多级权限管理（主人/管理员/普通用户）

### 3.2 操作相关

#### 命令面板
- `#帮助` / `#菜单` - 显示命令面板
- `#设置` - 显示设置选项
- `#资讯` - 显示资讯相关功能

#### 对话触发
- @提及触发 - 群聊中 @机器人触发对话
- 私聊触发 - 直接私聊机器人
- 命令前缀触发 - 使用 `#` 或 `/` 前缀触发命令
- 关键词触发 - 检测特定关键词自动回复

#### 游戏资讯功能
- `#资讯` - 显示最新资讯列表
- `#公告` - 显示最新官方公告
- `#活动` - 显示当前活动
- `#版本` - 显示版本更新信息

### 3.3 权限相关

#### 权限等级
| 等级 | 名称 | 权限范围 |
|------|------|----------|
| 0 | 超级管理员（主人） | 所有功能，包括系统设置 |
| 1 | 管理员 | 群管理功能、部分设置 |
| 2 | 普通用户 | 基础查询、对话功能 |
| 3 | 黑名单用户 | 禁止使用 |

### 3.4 对话相关

#### 主动消息推送
- 早间资讯推送（默认 08:00）
- 晚间热点推送（默认 20:00）
- 版本更新提醒
- 活动开始提醒

#### 大模型对话接入
- 支持 OpenAI 兼容协议
- 支持多模型切换
- 角色扮演设定（爱芮）

#### 敏感词过滤
- 关键词匹配过滤
- 正则匹配过滤
- 用户输入过滤
- 模型输出过滤

#### 对话记忆管理
- 短期记忆（滑动窗口）
- 会话管理（群/私聊隔离）
- 长期记忆（ChromaDB 向量存储）
- 对话压缩（摘要生成）

## 四、非功能需求

### 4.1 性能需求
| 指标 | 要求 |
|------|------|
| 消息响应时间 | 普通命令 < 2s，大模型对话 < 10s |
| 并发处理 | 支持 100+ 并发消息处理 |
| 可用性 | 99% 以上可用性 |

### 4.2 安全需求
- API Key 加密存储
- 用户隐私数据脱敏
- 权限隔离
- 审计日志
